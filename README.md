Cloud-Database

Key Features
  1. Eventual Consistency Model
  2. Delta Sync using Remote Differential Compression
  3. Replication
  4. Consistent Hashing

Supported Commands:
  1. CONNECT
  2. GET 
  3. PUT 
  4. SYNC
